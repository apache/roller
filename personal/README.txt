
Example Roller configuration and testing files

rollertask.sh          - can be used to run a Roller task on the command-line 
roller.xml             - example Roller context configuration file for Tomcat
mod_jk.conf            - example mod_jk.conf config file for Roller 
roller-for-resin3.conf - example Roller config file for Resin 3
roller.conf            - example Roller config file for Resin
sample.build.xml       - extra Ant tasks, callable from main build script
sample.ant.properites  - properties for extra Ant tasks
eclipse/               - example Eclipse project and classpath files for Roller
testing/               - data files for Roller unit testing
