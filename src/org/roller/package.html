Roller is split into a number of top level packages and these packages can be grouped as follows:

Business Layer
  • org.roller.model - Interfaces that define various Roller manager classes.
  • org.roller.pojos - Domain objects and other persistent objects.
  • org.roller.business - Impementations of the Roller org.roller.model interfaces.
Presentation Layer
  • org.roller.presentation - Roller's Web UI and Web Services implementations.
Other...
  • org.roller.util - General purpose utility classes, mostly independent of Roller.
  • org.roller.persistence - Persistence abstraction allows Roller to use either Castor or Hibernate.
